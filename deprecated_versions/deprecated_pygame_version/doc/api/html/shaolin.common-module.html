load_image(filename, datadir='./', convert=False)

source code 

Carga una imagen y le asigna un color de transparencia

La transparencia de la imagen depende del parámetro "convert", si convert vale False se tomará el canal alpha de la imagen. Si "convert" vale True se asigna un color transparente, de idéntico color del segundo pixel de la primer linea, el pixel (1, 0).
