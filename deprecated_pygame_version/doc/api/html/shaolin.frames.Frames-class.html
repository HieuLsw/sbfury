Class Frames

source code

Administra una collección de imagenes de un estado.

Cada imagen debe contener en su nombre la cantidad de cuadros que contiene la grilla completa, por ejemplo 'run_5.png' indica que la animación está compuesta por una grilla de 5 cuadros (de igual tamaño) separados por una linea horizontal a la izquierda

_get_frames_number_from_filename(self, filename)

source code 

Obtiene la cantidad de cuadros que tiene una grilla.

La cantidad de cuadros en una grilla se especifica en el mismo nombre de archivo, por ejemplo 'run_5.png' indica que la imagen del estado 'run' tiene 5 cuadros de animación.
